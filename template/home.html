This site automagically publishes Go package documentation for Github hosted projects. To add your project to this site, add http://gopkgdoc.appspot.com/hook/github to the post-receive hooks for your project. After adding the hook, click "Post-Receive URLs" and "Test Hook" to initiate the first update to this site.

File an issue to report bugs or suggest improvements. The source for this site is hosted on Github. Pull requests are welcomed.
