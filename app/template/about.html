About

GoPkgDoc displays documentation for Go packages on Bitbucket, Github, Launchpad and Google Project Hosting.

GoPkgDoc is hosted on Google App Engine. The source code available on Github.

Feedback

Send your ideas, feature requests and questions to gopkgdoc@beagledreams.com. Report bugs using the Github Issue Tracker.

Writing documentation for GoPkgDoc

GoPkgDoc produces documentation from Go source code. The guidelines for writing documentation for godoc apply to GoPkgDoc.

GoPkgDoc checks for package updates once per hour. You can force GoPkgDoc to refresh the documentation immediately by clicking the ‚ü≤ link at the bottom of the package documentation page.

Bookmarklet
